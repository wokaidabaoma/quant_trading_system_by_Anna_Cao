# 华尔街母鸡 - 技术指标完整使用指南

## 🎯 核心技术指标系统

### 1. RSI - 相对强弱指标 (胜率79.4%)
```
参数: RSI(14)
买入信号: RSI < 30 且开始上升
卖出信号: RSI > 70 且开始下降
背离信号: 价格新高但RSI未创新高(顶背离)
```

**实战技巧:**
- RSI < 20: 极度超卖，强烈买入信号
- RSI 30-50: 健康回调区间
- RSI 50-70: 上升趋势确认
- RSI > 80: 极度超买，考虑获利了结

### 2. 移动平均线系统 (EMA回报率1.60)
```
EMA(12, 26): 短期趋势
EMA(50, 200): 中长期趋势

金叉: 短期上穿长期(买入)
死叉: 短期下穿长期(卖出)
支撑/阻力: 价格回测均线
```

**操作策略:**
- EMA12 > EMA26 > EMA50 > EMA200: 完美多头排列
- 价格站上EMA50: 中期趋势转强
- EMA200: 牛熊分界线

### 3. 布林带 - Bollinger Bands (成功率77.8%)
```
参数: BB(20, 2) - 20日均线，2倍标准差
突破上轨+放量: 强势买入
跌破下轨+缩量: 超卖反弹
带宽收窄: 即将变盘
```

**实战应用:**
- 价格触及下轨 + RSI<30: 高概率反弹
- 价格突破上轨 + 成交量>2倍: 强势突破
- 布林带收窄<5%: 大行情即将来临

### 4. MACD - 双参数系统
```
标准MACD(12, 26, 9): 经典参数
快速MACD(3, 10, 16): Linda Raschke参数(更灵敏)

MACD金叉: 买入
MACD死叉: 卖出
柱状图由负转正: 动能转强
```

**高级用法:**
- MACD零轴之上金叉: 强势买入
- MACD背离: 趋势即将反转
- 快慢MACD共振: 高胜率信号

### 5. ATR - 平均真实波幅
```
参数: ATR(14)
止损设置: 入场价 ± 2×ATR
仓位调整: 高ATR减仓，低ATR加仓
突破确认: 突破幅度 > 1×ATR
```

**风险管理:**
- ATR > 5%: 高波动，减少仓位
- ATR < 2%: 低波动，可适当加仓
- 止损距离 = 2×ATR，止盈距离 = 3×ATR

## 📊 成交量分析系统

### 6. 成交量指标组合
```
Volume MA(20): 20日均量
Volume Ratio: 当日量/均量
OBV: 能量潮

放量突破: 量比 > 2.0
缩量调整: 量比 < 0.5
OBV与价格背离: 趋势反转
```

### 7. VWAP - 成交量加权平均价
```
价格 > VWAP: 偏强
价格 < VWAP: 偏弱
VWAP斜率: 趋势方向
偏离度 > 2%: 异常情况
```

**日内交易策略:**
- 开盘价格在VWAP之上: 做多为主
- 价格回测VWAP获得支撑: 买入机会
- 价格跌破VWAP: 谨慎操作

## 🚨 实时监控警报

### 成交量异动:
- 量比 > 3.0: 重大事件
- 量比 > 2.0 + 价格变动>3%: 异动警报
- 连续3日缩量<0.5: 变盘在即

### 技术信号:
- RSI极值(<20或>80): 极端情况
- MACD零轴金叉: 强势确认
- 突破关键阻力+放量: 追涨信号

### 风险警报:

- 价格跳空>5%: 重大消息
- 成交量枯竭<0.3倍均量: 趋势末期

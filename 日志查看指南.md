# 🐔 华尔街母鸡日志查看指南

## 📊 日志查看工具概览

华尔街母鸡系统提供了三种不同的日志查看工具，满足不同的使用需求：

### 🚀 快速查看工具
**适用场景**: 快速检查最新信号  
**使用方法**: `python3 quick_log.py`

### 🔧 命令行工具  
**适用场景**: 详细分析和统计  
**使用方法**: `python3 log_viewer.py [命令]`

### 🖥️ 交互式浏览器
**适用场景**: 友好界面，深度探索  
**使用方法**: `python3 interactive_log.py`

---

## 📋 详细使用方法

### 1. 快速查看 (`quick_log.py`)

最简单的查看方式，显示最新5条记录和今日统计：

```bash
python3 quick_log.py
```

**输出示例**:
```
🐔 华尔街母鸡 - 最新交易信号
==================================================
1. 🟢 RIOT   $  11.44 RSI: 26.5 | RSI超卖反弹
   📅 08-12 21:51 | 仓位: 1124股 ($12,859)
2. 🟢 COIN   $ 322.62 RSI: 24.0 | RSI超卖反弹
   📅 08-12 21:51 | 仓位: 52股 ($16,776)

📊 总记录数: 25
📅 今日: 25个信号 (🟢20 🔴5)
```

### 2. 命令行工具 (`log_viewer.py`)

#### 查看今日所有信号
```bash
python3 log_viewer.py today
```

#### 查看最新N条记录
```bash
python3 log_viewer.py latest 20    # 查看最新20条
python3 log_viewer.py latest       # 默认10条
```

#### 查看特定股票的历史
```bash
python3 log_viewer.py symbol MARA
python3 log_viewer.py symbol COIN
```

#### 查看统计分析
```bash
python3 log_viewer.py stats 7      # 最近7天统计
python3 log_viewer.py stats 30     # 最近30天统计
python3 log_viewer.py stats        # 默认7天
```

#### 查看表现分析
```bash
python3 log_viewer.py performance
```

#### 导出为CSV文件
```bash
python3 log_viewer.py export                    # 默认文件名
python3 log_viewer.py export my_signals.csv     # 自定义文件名
```

### 3. 交互式浏览器 (`interactive_log.py`)

启动交互式界面：
```bash
python3 interactive_log.py
```

**功能菜单**:
```
📋 功能菜单:
  1. 查看今日信号      ← 今日所有交易信号
  2. 查看最新记录      ← 最新N条记录
  3. 按股票查询        ← 特定股票的历史
  4. 统计分析         ← 详细统计信息
  5. 表现分析         ← 盈亏模拟分析
  6. 时间范围查询      ← 自定义时间区间
  7. 导出数据         ← 导出为CSV
  8. 刷新日志         ← 重新加载日志
  9. 退出
```

## 📊 日志内容解读

### 信号显示格式
```
序号. 🟢/🔴 股票代码 $价格 RSI:数值 | 信号原因 | 时间
```

- **🟢**: 买入信号
- **🔴**: 卖出/做空信号  
- **⚪**: 无信号
- **RSI**: 相对强弱指数
- **信号原因**: 
  - `MACD金叉`: MACD线突破信号线
  - `RSI超卖反弹`: RSI<30后反弹
  - `RSI超买回落`: RSI>70后回落
  - `布林带突破`: 价格突破布林带上轨

### 详细信息包含
- **价格**: 触发信号时的股价
- **技术指标**: RSI、成交量比率、ATR等
- **仓位建议**: 建议买入股数和金额
- **风险控制**: 止损价位和止盈目标
- **内幕交易**: 相关内幕交易信息（如有）

## 🔍 故障排除

### 常见问题

#### 1. "日志文件不存在"
**原因**: 还没有运行过扫描  
**解决**: 先运行一次扫描 `python3 main.py once`

#### 2. "无数据可查看"
**原因**: 日志文件为空或损坏  
**解决**: 检查 `logs/trading_signals.log` 文件

#### 3. 导出失败
**原因**: 权限问题或磁盘空间不足  
**解决**: 检查目录权限和磁盘空间

#### 4. 统计数据不准确
**原因**: 日志文件可能有格式错误  
**解决**: 使用交互式浏览器的"刷新日志"功能
